Scroll Class Plugin Demo

The header (and the h1 tag) gets .scroll after you scroll beyond 32px. Note scroll-threshold inherits. This fits with the Clik system where we only really want one scroll state

This page demonstrates applying classes to divs on scroll using scrollFix. The divs need a class (.scroll-fix) applying to them to trigger the behaviour. Otherwise we'd have to check every container and every CS.

As all this does is apply a class (scroll by default) you can control all the bevahiours via CSS anyway.
