# Stylesheet selection plug-in

Write a JavaScript plugin in to allow people to select different stylsheets for a site. I am happy to use Jquery but would be interested to see how a native version compares. I need modal and draggable functionality and don't want to bloat the code.

I will kick off two simultaneous versions and see how they compare.

The plug in will

- Show an icon in the top right of the screen. It will be draggable to get it out of the way
- When you click on it, it will open a list of available stylesheets. These will have a title and a decriptions. An optional preview image can be defined and this will show on hover in a panel to the left
- The data for the styles will be in Json, create a sample JSON file to go with the data. The plug in should load this
- On selecting the style, the site will change to that stylesheet
- I would like somehow to persist the choice of style in perhaps local storage
